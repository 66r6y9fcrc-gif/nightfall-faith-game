# 🩸 Nightfall Faith

> Ein düsteres Gothic-Metroidvania Mobile Webspiel — inspiriert von The Last Faith.  
> Spielbar direkt im Browser, ohne Installation.

## 🎮 Jetzt spielen

👉 **[nightfall-faith-game spielen](https://66r6y9fcrc-gif.github.io/nightfall-faith-game/last-faith-style-mobile-game.html)**

## Steuerung

| Plattform | Bewegen | Angriff | Dash |
|---|---|---|---|
| **Handy / Tablet** | Virtueller Stick (links) | ATK-Knopf (rechts) | DASH-Knopf (rechts) |
| **Desktop** | WASD / Pfeiltasten | Leertaste / J | Shift / K |

## Gameplay

- Überlebe so lange wie möglich gegen immer mehr Bestien
- Sammle **Schattenessenz** durch das Besiegen von Gegnern
- Nutze den **Schatten-Dash** zum Ausweichen — du bist dabei kurz unverwundbar
- Wenn deine HP auf 0 fallen, kannst du einen neuen Run starten

## Tech

- Reines HTML5 + Canvas + Vanilla JS — kein Framework, keine Dependencies
- Läuft auf Handy, Tablet und Desktop
- Touch-optimiert mit Pointer Events API
